# Product

## Register

product

## Platform

web

## Users
Support operators, sales agents, and automated AI assistants handling customer voice calls over WhatsApp. The primary audience requires real-time, low-latency, focus-oriented dialers and CRM widgets to manage active conversations efficiently. The secondary audience is system administrators configuring sessions, credentials, webhooks, and automated AI scheduling.

## Product Purpose
WhatsApp VoIP voice calling directly from the browser, supporting multi-account session management, API integrations, and CRM widget embedding (e.g. Chatwoot). Success means low-latency VoIP connections, crystal-clear real-time call states, and reliable automated calling campaigns.

## Positioning
Direct browser-to-WhatsApp VoIP calling with seamless, autonomous server-side AI voice agents integrated with CRM tools.

## Brand Personality
Professional, robust, tech-forward, and highly reliable. Emphasizes visual clarity, clean UI states, and functional efficiency to handle real-time audio communications.

## Anti-references
Avoid generic SaaS dashboard clichés, such as excessive card shadows paired with borders, over-rounded corners (24px+) on inputs/cards, tiny tracked uppercase eyebrows above every layout section, or redundant metric grids.

## Design Principles
- **Real-Time State Clarity**: The interface must reflect call progress, audio status, and active connections instantly, without any visual lag or state ambiguity.
- **Compact CRM Harmony**: Embeddable widgets must feel integrated, compact, and non-disruptive to the host CRM platforms (like Chatwoot) while retaining full control.
- **High-Frequency Efficiency**: Design dials, lists, logs, and toggles for high-frequency day-to-day use, prioritizing focus and reducing cognitive load.

## Accessibility & Inclusion
Standard accessibility (WCAG 2.1 AA target) featuring clear color contrast for active/inactive call states, a fully keyboard-navigable dialer, and descriptive ARIA labels for audio/mute controls. Support for reduced motion to toggle off decorative transitions when configured.
